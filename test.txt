• 리눅스 명령어
– ls, pwd, cd, touch, mkdir, rm 등 몇 가지 명령어를 사용
• GIT 저장소(Repository)
– 컴퓨터에 설치하는 git 작업 공간 또는 저장소가 필요
– 저장소는 repository 또는 rep라고 한다. 
• GIT이 활성화되어 있지 않다면
– 내 컴퓨터(local)에 저장할 곳이 필요
– (중요) GIT에게 언제 사용하고, 어떤 폴더(directory)에 저장소를
만들어야 하는지 알려줘야 한다.
• GIT을 사용하고 싶을 때마다 또는 새로운 프로젝트 마다 새로운
저장소를 생성할 수 있다. 
– 고유한 기록이 있는 git 작업 공간을 의미
